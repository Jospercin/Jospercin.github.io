Aplicacion simple de un registro en AngularJS con distintas funciones basicas.

- Preferiblemente usar con navegador Chrome
- Navegador Mozilla y Otros necesita una edicion en el diseño del formulario.

1º Página (index2.html):

Registro de Usuario: 
    - Nombre
    - Contraseña
    - Email

Opcion de Registro o de Reinicio de los campos (No estan controlados todos los errores 
para una funcionalidad 100% correcta) Todos los campos son requeridos para el 
registro dentro del formulario.

Seleccion de ¿Tiene ya una cuenta? = Redireccion 2º página (index.html)

2º Página (index.html):

- Select simple con opciones de distintas asignaturas del modulo de DAW

- Sorpresa : Boton con funcion de AngularJS que permite mostrar un mensaje 
al momento del click (o de ocultarlo si no esta clicked).

- Añade un click: Boton que por cada click aplicado muestra un listado en el inferior en orden
descendente. (Ejemplo 50 Clicks = 50 elementos en la lista + los 4 iniciales => 54 elementos totales)


- Pequeño o grande: Dependiendo del boton clickeado el texto inferior cambiara al tamaño indicado.

- Hola ¿Cómo estás?: Click para Redireccion 1º página (index2.html)
